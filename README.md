# _Научиться учиться_

Проект посвящен проблемам и популярным методам обучения. Цель проекта - замотивировать пользователя узнать о понравившихся подходах к обучению и предложить несколько ресурсов для самостоятельного развития.

Для визуализации контента использованы технологии Flexbox и animation.

В планах заменить Flexbox на grid для размещения карточек. Добавить анимационных эффектов.
